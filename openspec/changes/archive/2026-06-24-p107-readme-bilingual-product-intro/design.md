# P107 Design

The root `README.md` remains the English GitHub landing page. A new `README.zh-CN.md` provides a Chinese mirror with the same public-facing structure and safety posture.

The language switch is plain Markdown:

- `English | 简体中文` in the root README.
- `简体中文 | English` in the Chinese README.

The Chinese README intentionally avoids adding stronger claims than the current product and release evidence supports. It describes Investment Agent as a local-first research, portfolio maintenance, evidence review, and manual decision governance cockpit.
